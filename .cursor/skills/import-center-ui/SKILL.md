# Skill: Import Center UI

## When to use
- Import center UX/UI tasks
- Upload/mapping/validation/fix-flow interactions

## Objective
Make Import Center a complete operational flow, not a simple upload form.

## Required UX Flow
1. Create import job
2. Upload/register source
3. Map source fields to target fields
4. Run validation
5. Show issue details
6. Provide one-click fix entry
7. Return to same context after fix

## Required UI Blocks
- Job queue/status strip
- Mapping editor area
- Validation issue list with severity
- Fix actions and quick links
- Job history / audit timeline

## Anti-patterns
- Single upload button with no mapping state
- Non-actionable error text
- No traceability for source and fix operations
